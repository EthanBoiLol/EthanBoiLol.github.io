Relight Engine

Relight Engine is a free and open source reimplementation of Unreal Engine via reverse engineering

Relight Team | About us

The Relight team was created in May 2024 by EthanBoiDev. The goal of this team is to reverse engineer Unreal Engine without copying any source code from UE. This project started when Ethan realized that the ELUA for Unreal Engine allows reverse engineering for "competing engines" so long as no code is copied, thus, Ethan started the project to create a complex engine with unreal-like technology for everyone to enjoy
