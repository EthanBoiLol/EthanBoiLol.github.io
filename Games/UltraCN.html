Ultra Custom Night: Open Source

Status: Protoype [Hiatus]

Ultra Custom Night: Open Source, also known as UltraCN: Open Source, is a FOSS version of the fnaf fangame called "Ultra Custom Night" to GDevelop

Check out the original by KamilFirma!

Imagine if Ultimate Custom Night had 700+ characters? Do you think you can take all of them? Ulrta Custom night, created by KamilFirma, tries to answer this question, by added almost every single characters and animatronics from the series. UlraCN: Open Source aims to rewrite the entire game from Clickteam to GDevelop while keeping the game itself open source
